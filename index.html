Annotation demo

This is the demo of a Javascript plugin that allows simple annotations on any page.

Requirements

  • Browsers: IE9+, Chrome, Firefox, iPad
  • Libraries: Zepto / jQuery, Bootstrap

Usage

  • Click on the annotate button to start drawing.
  • Select an item to edit it. You can only change its size, not move it.
  • After selecting, press Esc to delete it.

Installation

Add this line to your script

<script class="annogram" src="annotate.js"></script>

IMPORTANT:

  • The class=annogram should not be changed.
  • Zepto / jQuery, and bootstrap.min.css should be part of the main page.
